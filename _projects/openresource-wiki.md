## Overview

**OpenResource Wiki** is a community-driven, open-source knowledge base that curates practical AI tools, prompt engineering techniques, and agent skill frameworks. Originally shared through Xiaohongshu (Little Red Book) posts, the project grew into a structured wiki hosted on GitHub Pages.

**Live site:** [openresource-wiki](https://gy-hou.github.io/openresource-wiki/)

## What's Inside

| Category | Description |
| --- | --- |
| AI Tools | Curated reviews and usage guides for popular AI applications |
| Prompts | Ready-to-use prompt templates for various LLM tasks |
| Agent Skills | Reusable skill definitions and workflows for AI agents |

## Motivation

High-quality AI resources are scattered across platforms and often behind paywalls. This project consolidates practical, tested knowledge into a single open-access wiki — free for everyone to use and contribute to.

## Contributing

Contributions are welcome via pull requests or GitHub Issues. Whether it's a new tool review, a refined prompt, or a bug fix, all contributions help the community.
